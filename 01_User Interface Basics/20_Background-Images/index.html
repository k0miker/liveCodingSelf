Hintergrund-Bilder

Neben dem img-Element gibt es noch einen zweiten Weg wie wir Bilder zu unserer Webseite hinzufügen können und das ist als Hintergrund-Bild.

Dazu gibt es die background-image Eigenschaft in CSS (bzw dessen Abkürzung background).

Mit background-image bitte nur dekorative Bilder zur Seite hinzufügen, denn ein Screen-Reader kann ein background-image nicht auslesen. Für inhaltliche Bilder bitte immer das img-Element benutzen.

Einstellungen:

  • background-image: Verlinkt zum Bild
  • background-size: Gibt an wie das Bild den Hintergrund ausfüllen soll (Typische Werte sind cover, contain oder Größenangaben in Prozent oder Pixel)
  • background-repeat: Gibt an ob das Bild wiederhollt werden soll, wenn freier Platz ist (Werte: repeat oder no-repeat)
  • background-position: Definiert die Position des Bildes im Hintergrund
