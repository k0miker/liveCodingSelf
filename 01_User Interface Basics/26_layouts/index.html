Layout-Tipps

  • Die Höhe von Elementen nur setzen, wenn es absolut notwendig ist. Ein Element sollte sich an den Inhalt anpassen können.
  • position: relative nur für kleine Änderungen der Position nutzen
  • Wenn möglich (& sinnvoll) relative Einheiten verwenden, damit das Layout flexibler ist
  • Testet euer Layout immer wieder, indem ihr Textgrößen verändert, Text-Inhalte verändert oder die Breite des Browser-Fensters anpasst
  • Komplexere Layouts benötigen Container-Boxen. Nehmt euch die Zeit das Layout zu planen und sauber aufzubauen.
