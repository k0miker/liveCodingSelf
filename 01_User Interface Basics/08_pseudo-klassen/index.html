Pseudoklassen

Mit Pseudoklassen können wir CSS-Regeln auf einen bestimmten Status von Elementen anwenden

:hover

Mit der :hover Pseudoklasse können wir ein Element stylen, wenn sich die Maus über diesem befindet.

:active

Die :active Pseudoklasse wird aktiviert, wenn ein Link geklickt wird (während des Klicks):

Active Link

:visited

Mit der :visited Pseudoklasse können wir das Aussehen eines Links definieren, nachdem dieser besucht wurde:

DuckDuckGo

Reihenfolge der Pseudoklassen

Falls ein Element mehrere Pseudoklassen bekommt, dann muss das Styling in der folgenden Reihenfolge stattfinden:

  1. :visited
  2. :hover
  3. :active
