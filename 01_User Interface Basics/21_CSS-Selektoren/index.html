CSS-Selektoren

  • div p : Alle Elemente vom Typ p, die sich innerhalb von einem Element vom Type div befinden.
  • div > p : Alle p-Elemente, die als direktes Eltern-Element ein div-Element haben (bzw. direkte Kinder von einem div sind).
  • div + p : Alle p-Elemente, die direkt nach einem div-Element positioniert sind. (Geschwister-Element und direkt folgend)
  • div ~ p : Alle ul-Elemente, die nach einem div-Element kommen. ul-Elemente müssen Geschwister von einem div-Element sein und nach diesem positioniert sein.
  • d..iv p:nth-child(2) : Alle p-Elemente, die sich innerhalb eines div befinden (egal wie tief) & die das zweite Kind ihres Eltern-Elementes sind.

Es gibt auch noch weitere Selektoren wie :first-child, :last-child oder :nth-of-type(). Eine Übersicht über alle Optionen findet ihr hier
