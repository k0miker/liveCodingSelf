CSS Soft Reset

Das Ziel des Soft Reset im CSS ist es sicherzustellen, dass das Layout, welches wir erstellen möglichst nah an unsere Angaben herankommt. Es gibt zwei Herausforderungen dabei:

1. Alle Elemente (inkl. Body) haben bereits Margin & Padding-Einstellungen vom Browser

Daher macht es Sinn für uns die Margin- & Padding-Einstellungen für alle Elemente auf der Seite zurückzusetzen. So können wir die Einstellungen selektiv & kontrolliert einzeln überschreiben.

2. Die Größe von Elementen wird vom CSS standardmäßig exklusive von Border & Padding berechnet. Das ist unintuitiv & unpraktisch.

Wenn wir die Größe von Elementen setzen, wollen wir das die Breite eines Elementes inklusive von Border & Padding berechnet. Das heißt wenn ich ein Element mit 100px Breite definiere & es mit Padding und Border versehe, soll es trotzdem insgesamt nur 100px groß auf der Seite sein.

Woran liegt das?

Element haben standardmäßig die Einstellung box-sizing: content-box. Wir nutzen lieber die Einstellung box-sizing: border-box

Diese content-box hat eine Gesamtbreite von 190px: 150px width + 20px Padding + 20px Border.

box-sizing: content-box; width: 150px

Diese border-box haht eine Gesamtbreite von 150px (inkl 20px Padding & 20px Border).

box-sizing: border-box; width: 150px

Lösung für das Problem

Fügt einfach den folgenden Code ganz nach oben zu eurem CSS hinzu, damit ihr den Soft Reset durchführt:

* { margin: 0; padding: 0; box-sizing: border-box; }
