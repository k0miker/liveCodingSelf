Positions-Layout

Mit der CSS-Eigenschaft position können wir die Positionierung eines Elements beeinflussen.

position: static

Der Standard-Wert der Position. Element normal im Flow-Layout positionieren.

position: relative

Wir verschieben das Element mit top, left, right & bottom relativ zur eigentlichen Position im Flow-Layout. Die Verschiebung hat keinen Einfluss auf die Position der anderen Elemente auf der Seite.

Mit dem z-index können wir die Reihenfolge des Renderings definieren. Je höher der Wert ist, desto weiter vorne ist das Element. Der Standardwert ist 0

position: relative; top: 4rem; left: 120px;
position: relative; bottom: 60px; right: 60px;
position: relative; bottom: 80px; left: 60px;

position: absolute

Mit position: absolute können wir Elemente relativ zu den Grenzen eines Containers positionieren.

Der Container ist das nächste Eltern-Element mit position: relative.

Das Element wird dann aus dem Flow der Seite genommen. Nicht benutzen, wenn ein Element im Flow positioniert sein soll.

Mit top, left, right & bottom könnt ihr den Abstand zu den Seiten des Containers definieren.

position: relative;
position: absolute; top: 64px; right: 64px; left: 16px; bottom: 16px;
position: absolute; top: 16px; right: 20%;

Beispiele für Anwendungen:

  • Modals, Popups (Elemente über dem Inhalt der Webseite)

position: fixed

Das Element wird im Browser-Fenster fixiert.

Mit top, left, right & bottom könnt ihr den Abstand zum Rand des Browser-Fensters definieren.

position: fixed; top: 0; left: 0;
position: fixed; bottom: 16px; left: 16px; right: 16px;

position: sticky

Das Element bleibt am Bildschirmrand in definiertem Abstand 'kleben'.

Mit top, left, right & bottom könnt ihr den Abstand zum Fensterrand definieren.

position: sticky; top: 120px; bottom: 64px;
