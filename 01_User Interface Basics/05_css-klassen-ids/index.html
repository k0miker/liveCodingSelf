CSS-Klassen & IDs

Klassen & IDs sind zusätzliche Wege wie wir CSS-Stile auf HTML-Elemente anweden können. IDs und Klassen können wir auf alle HTML-Elemente anwenden.

Klassen

Mit einer Klasse kann ich einen CSS-Stil auf so viele Elemente wie ich möchte anwenden. Ich kann den Stil wiederverwenden.

IDs

Mit einer ID können wir ein spezifisches Element auf der Seite identifizieren. Eine ID muss eindeutig sein - d.h. nur ein Element darf eine bestimmte ID haben. Mittels CSS können wir dann Stil-Regeln direkt auf dieses Element anwenden.
