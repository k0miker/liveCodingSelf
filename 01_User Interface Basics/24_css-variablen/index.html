CSS-Variablen

Sobald wir Werte in unserem CSS mehrmals verwenden möchten, macht es Sinn diese in einer Variablen zu speichern. So ist der Wert im CSS nur einmal definiert und kann ganz einfach geändert werden.

Dazu haben wir CSS-Variablen.

Definition von CSS-Variablen: --variablen-name: wert;

Nutzung von CSS-Variablen: var(--variablen-name)

Typische Anwendungsfälle:

  • Definition von Farben bzw. der Farbpalette
  • Schatten
  • Farbverläufe
  • ...
