Flexbox-Elemente

Es gibt eine Reihe an Einstellungen für die Flexbox-Elemente, mit denen wir deren Verhalten und Position beeinflussen können.

flex-basis

flex-basis definiert die Anfangsgröße eines Elementes. Der Standardwert ist auto. Diese Größe ist nicht fix (wie bei width/height), sondern reagiert auch auf die flex-grow & flex-shrink-Einstellungen.

flex-basis ist die Größe entlang der Hauptachse.

flex-basis: 100px;
flex-basis: 30%;
flex-basis: 10%;

flex-grow

Bestimmt, wie schnell das Element wachsen soll, wenn zusätzlicher Platz verfügbar ist. Je höher der Wert, desto mehr Platz nimmt das Element in Anspruch. Der Standardwert ist 0.

flex-grow: 1
flex-grow: 0
flex-grow: 1
flex-grow: 3
flex-grow: 1
flex-grow: 0

flex-shrink

Bestimmt, wie schnell das Element schrumpfen soll, wenn nicht genügend Platz vorhanden ist. Je höher der Wert desto schneller schrumpft das Element. Der Standardwert ist 1.

flex-shrink: 0;
flex-shrink: 1;
flex-shrink: 2;
flex-shrink: 0;
flex-shrink: 1;

flex-Abkürzung

flex: [flex-grow] [flex-shrink] [flex-basis]

Mittels flex können wir die drei Werte flex-basis, flex-grow & flex-shrink zusammen setzen

order

Mit order können wir die Reihenfolge der Flex-Elemente ändern. Der Standardwert ist 0. Je höher der Wert ist, desto weiter hinten ist das Element angeordnet.

order: 0
order: 2
order: 0
order: -2
order: -1

align-self

Mit align-self können wir das align-items-Verhalten für ein einzelnes Element überschreiben.

align-self: flex-end
align-self: center
align-self: stretch
