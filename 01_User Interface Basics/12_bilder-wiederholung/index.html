Das Alt-Attribut

Das Alt-Attribut ist eine textliche Beschreibung des Bildes & wird dargestellt, wenn das Bild nicht geladen wird. Suchmaschinen nutzen das Alt-Attribut ebenfalls für SEO.

Das Title-Attribut

Das Title-Attribut ist der Titel des Bildes. Dieser wird angezeigt, wenn wir mit der Maus über das Bild hovern.

Die Größe eines Bildes setzen

width & height-Eigenschaften

Wir können die Größe eines Bildes mit width & height definieren. Alle Größeneinheiten sind nutzbar hier.

auto ist ein spezieller Wert für Breite & Höhe. Damit wird die Größe automatisch an das Format des Bildes angepasst. auto stellt sicher, dass nichts verzerrt wird.

Wie implementieren wir eigene Größenformate ohne Verzerrung?

object-fit: cover;

Das Bild füllt den Container aus

object-fit: contain;

Das komplette Bild wird innerhalb des Containers angezeigt.

Bildformate

  • JPG/JPEG: Standardformat für Fotos mit viel Detail. Pixel-basiert. Ohne Transparenz
  • PNG: Bilder mit Transparenz. Pixel-basiert.
  • SVG: Vektor-Dateien. Verlustfreie Darstellung von Logos, Icons & Vektor-Grafiken
  • GIF: Ein kurzes loopendes Video. Stark komprimiert.
  • WEBP: Für das Web optimierte Bild-Datei. Extra starke Konvertierung. (Alternative zum JPG)
  • ICO: Format fürs Favicon
