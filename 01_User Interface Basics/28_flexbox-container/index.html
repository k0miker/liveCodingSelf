Flexbox

Die CSS Flexbox ist ein Layout-Modell, das die Struktur von Boxen in einem Container optimiert. Die Flexbox bietet eine flexible Möglichkeit Layouts zu gestalten und ist besonders nützlich für responsives Design.

Mit der Einstellung display: flex machen wir aus einem Container eine Flexbox:

Unsere erste Flexbox

Innerhalb der Flexbox können wir ganz einfach eine variable Anzahl an Elementen darstellen. Die Art und Weise der Anordung definieren wir mit den Einstellunge flex-direction & flex-wrap.

Mit flex-flow können wir flex-direction & flex-wrap zusammen setzen.
