CSS-Einführung

Mittels CSS können wir die Inhalte unserer Webseite stylen.

Es gibt drei verschiedene Wege CSS zu integrieren:

  • Internes CSS
  • Externes CSS
  • Inline CSS
Externes CSS ist der Standard-Weg, um CSS zu integrieren. Wir wollen CSS & HTMl voneinander getrennt halten.
Inline CSS

Man kann CSS auch inline direkt zu HTML-Elementen hinzufügen. Diese CSS-Regel har dann oberste Priorität. Wir wollen Inline CSS aber gar nicht (!) benutzen.
