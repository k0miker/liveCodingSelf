Farben in CSS

Im CSS haben wir verschiedene Möglichkeiten Farben zu definieren.

In diesem Artikel sind alle Optionen beschrieben: Color Formats in CSS

Benannte Farben

Es gibt eine Liste an benannten Farben, die wir in CSS nutzen können. Dazu müssen wir nur den Namen der Farbe referenzieren. Alle Farben findet ihr in der Dokumentation: MDN Dokumentation

RGB-Definition

Alle Pixel auf einem Bildschirm bestehen aus Rot-, Grün- & Blau-Anteilen. In CSS können wir rgb(255 255 255) nutzen, um diese drei Farb-Anteile zu definieren. Die Werte sind jeweils von 0 bis 255.

Hex-Code-Definition

Der Hex-Code ist einfach nur eine andere Schreibweise für die RGB-Definition.

  • #FF00FF => rgb(255 0 255)
  • #FF00FF00 => rgb(255 0 255 / 0)

HSL-Definition

Die HSL-Definition ist ein weiterer Weg, wie Farben definiert werden können. Wir definieren hier den Farbton (Hue), die Sättigung (Saturation) & die Helligkeit (Lightness) einer Farbe. Diese Definition macht es uns einfacher Farbpaletten zu definieren.

Farb-Transparenz

Die Transparenz lässt sich direkt in der Farbdefinition mit definieren. Der Wert ist 0-1 (0 -> 0% Transparenz & 1 -> 100% Transparenz).

  • rgb(200 100 89 / 0.5) => 50% Transparenz
  • hsl(200 100 89 / 0.2) => 20% Transparenz
