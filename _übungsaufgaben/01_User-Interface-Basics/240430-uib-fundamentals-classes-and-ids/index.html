Classy and identified

IDs: IDs sind dazu da, um ein einzelnes Element auszuwählen. Dieses muss einzigartig sein.

Beispiele für einzigartige Elemente auf der Seite: Einzigartiger Header, Navigation oder andere einzigartige Elemente...

Klassen: Klassen sind dazu da, um mehrere Elemente auf einmal zu addressieren. Mit Klassen CSS-Stile wiederverwenden.

Mit .test-style kann man zum Beispiel alle Elemente mit der Klase "test-style" stylen.
