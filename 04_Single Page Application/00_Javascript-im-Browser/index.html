Javascript im Browser

JS-Skripte können grundsätzlich zum Head oder zum Body einer Seite hinzugefügt werden. Die Skripte werden der Reihenfolge nach von oben nach unten ausgeführt.

Ein Skript im Head wird standardmäßig ausgeführt bevor HTML-Elemente geladen wurden. Ein Skript am Ende des Bodys wird ausgeführt, nachdem die HTML-Elemente geladen wurden.
